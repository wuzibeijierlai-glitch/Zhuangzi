## 缘起

今日之世人，大多行色匆匆，心力交瘁。我们为了名利、为了虚无的死后荣耀在烈日下拼命狂奔，犹如那个“畏影恶迹”的愚人——越是奔跑，脚印越多；越是挣扎，阴影越深，最终只能在精神的荒漠中力竭而死。

孟子曾说：**“颂其诗，读其书，不知其人，可乎？是以论其世也。”**
市面上绝大多数的《庄子》读本，往往只做到了生硬刻板的“字”的对译，却彻底忽略了字面背后“人”的温度。庄子不仅是一位哲人，更是一位活生生、在陋巷里饿得面黄肌瘦，却依然能把灵魂托付给璀璨星空、与造物者优游相处的千古天才。

这部《庄子全书 · 无障碍沉浸版》的诞生，不为标榜，不求名利，只为给您提供一处**“处阴以息影，处静以息迹”**的清凉树荫。我们在字里行间注入了生命的温度：
* 它的译文里有**“怒”**——那是对儒家虚伪礼教、做秀表演和世俗名利场极其无情、彻底的痛斥与解构；
* 它的解读里有**“悲”**——那是对芸芸众生“利禄是殉、易性而无悔”、被世俗面具死死扣住肉体灵魂的极度怜悯；
* 它的骨子里有**“安祥”**——在每一个喧嚣篇章的结尾，都巧妙而开阔地收束于“与大自然规律和解，与宇宙大道优游徘徊，获得生命的终极救赎”。

## 意指

在这只“虚舟”上，您将体验到与现代技术完全融合的大道之美：

### 🌿 法天贵真，不拘于俗
* **赤子之译**：翻译不玩文字游戏，不作学术堆砌。它用充满生命张力与思辨力度的语言，直接刺破千年的迷雾，直视人性深处的焦虑与局限。
* **知音之解读**：融入现代心理学、认知科学、生理学与存在主义哲学，它不教你如何“成功”，只教你如何“复归于真”，在险恶的人世间保全内心的完整与大宁。
* **圆融无缺**：全书三十二篇、二百二十七个区块，全部经由人工逐句深度打磨，绝无一处敷衍，绝无半个占位符。

### 🎨 望之似木鸡，德全而不耀
* **宣纸温润之色**：精心调配的古典宣纸暖色（`#f7f3ec`）与黛黑文字，最大程度保护双眼。深夜研读时，一键切入“大荒深林”般的暗黑模式，如落叶归根，温润无感。
* **素木宋体排版**：采用 Google Fonts 顶级衬线体 `Noto Serif SC`（思源宋体），字里行间如木刻版画，留白舒缓，呼吸自然。
* **独行而不改**：整个项目**仅由一个单文件 HTML 构成**。零框架、零数据库、零外部脚本。无依赖，双击即开。它就像一块路边的石头，无知无觉，却最能经受岁月的磨砺。

### ⚙️ 触字即弹，息影处静
* **分栏对照，心斋坐忘**：今译、原文、解读分栏并列，支持自由切换。在阅读原文时，手指轻触任何生僻字，即刻浮现清澈的注解，如水波漾起，旋即归于平静，绝不打扰您的沉浸体验。
