# 庄子全书 · 无障碍沉浸版

本项目将完整的《庄子》（内篇七、外篇十五、杂篇十，共三十二篇，二百二十七章）重构为单文件离线网页，方便读者免去网络和数据库依赖进行日常阅读与学习。

---

## 项目初衷

市面上许多《庄子》读本偏向于字句对译，有时难以直接呈现文字背后的思想力度。本项目旨在提供一个集“今译、原文、解读”于一体的无障碍阅读工具。在解读与今译中，融入了对庄子生命哲学（如重生轻物、心有天游、物我两忘等核心概念）的现代视角与存在主义心理学释义，力求拉近经典同现代读者的距离。

---

## 功能与设计

### 1. 内容阐释
* **今译与深度解读**：全书三十二篇、227 个章节均包含了详尽的翻译和结合现代社会困境的深度剖析，力求直面人性深处的焦虑与局限。
* **生僻字注音释义**：在阅读原文时，轻触生僻字即可悬浮显示拼音和详尽释义。

### 2. 视觉与排版
* **宣纸温润配色**：提供古典宣纸色（暖白 `#f7f3ec`）与深林（暗色模式）两种主题，以适应不同光线环境下的长读需求。
* **衬线宋体排版**：正文使用 Google Fonts `Noto Serif SC`（思源宋体）渲染，字距与行距均经过调试，以提供舒适的视觉体验。
* **单文件离线化**：整个项目由单个离线 HTML 文件构成，不依赖外部服务器、网络脚本和外部数据库，双击即可在任何现代浏览器中快速打开。

### 3. 对照交互
* **行级并排锚定布局**：今译、原文、解读支持分栏并列。针对古文精练、译文常带注解导致高矮不一的痛点，首创“行级并排锚定布局”，将古今句子封装在相同的网格行中，保证头部绝对水平对齐。
* **句级双向互锁高亮**：在“对照”标签下，轻触或滑过某句古文，对应的今译句子会自动亮起琥珀金背景；反之亦然。

## 对齐算法说明 (DP-IDF Sentence Aligner)

由于文言文极其精练，而今译本往往包含括号注解且信息量较大，传统的位置等比例映射或常规长度对齐算法（如 Gale-Church）在面临较大的长短句差异时容易产生对齐错位。

本项目在客户端浏览器中实现了一套基于**单调性全局动态规划与 IDF 相似度**的句子对齐算法（DP-IDF Sentence Aligner）。

### 1. 字符 IDF 统计与降权
算法在惰性渲染时，过滤掉标点符号，仅保留汉字：
* 对“之、乎、者、也、矣、而、于、以、其”等无语义高频文言助词降权（赋予 `0.05` 权重）。
* 统计当前章节内所有字符的文档频率 $\text{charFreq}[c]$。
* 为低频核心实体字（如人名、地名、核心名词概念）赋予更高的逆文档频率（IDF）权重：
  $$W(c) = 1 + \ln\left(\frac{N_{total}}{\text{charFreq}[c]}\right)$$

### 2. 加权交集相似度
计算原文句子 $U$ 与译文句子 $V$ 之间的非对称加权 Jaccard 相似度 $S(i, j)$，这能有效降低字数长短不一对匹配的干扰，精准定位字词关联：
$$S(i, j) = \frac{\sum_{c \in U \cap V} \min(W_U(c), W_V(c))}{\sum_{c \in U \cup V} \max(W_U(c), W_V(c))}$$

### 3. 全局单调性动态规划
基于“古文阅读顺序与今译序列单调非递减”这一天然先验，算法使用动态规划求解全局最优对齐轨迹，杜绝交叉错位：
* **状态定义**：$dp[i][j]$ 表示前 $i$ 个原文句子与前 $j$ 个今译句子匹配时的全局最大累计相似度。
* **状态转移方程**：
  $$dp[i][j] = S(i-1, j-1) + \max_{0 \le k \le j} dp[i-1][k]$$
* **逆向回溯**：利用记录转移路径的 `back` 数组逆向回溯，生成原文句子到译文句子的一对一/多对一的映射。

该算法在客户端浏览器环境下的运算耗时通常小于 1ms，保证了“对照”视图开启时的即时响应。

## 使用说明

### 离线双击（推荐）
1. 下载或克隆本项目：
   ```bash
   git clone https://github.com/wuzibeijierlai-glitch/Zhuangzi.git
   ```
2. 直接双击目录下的 `庄子全书_无障碍版.html`，即可在浏览器中开始阅读。

### 本地轻量 Web 服务器
若需要在局域网内或移动设备上游览，可在目录下启动 Python 轻量服务器：
```bash
python -m http.server 8000
```
访问 `http://localhost:8000/庄子全书_无障碍版.html` 即可。
